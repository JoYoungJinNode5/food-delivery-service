# food-delivery-service

## 프로젝트 소개
프로젝트명:  Express.배달
프로젝트 소개: 신속하고 정확한 음식 배달 서비스를 목표로 소비자와 사업자 모두를 만족하는 웹을 구현하였습니다.
사용자는 가게와 메뉴를 보고 배달 주문을 할 수 있으며, 사장님은 백오피스를 통해 해당 가게의 정보를 관리할 수 있습니다.

## 팀 소개
팀명: 3층 아키텍처 (Node.JS 5기 3조)
팀장: 조영진
팀원: 황소은, 김노을, 노시헌, 김현지
